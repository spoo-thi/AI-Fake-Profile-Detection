# AI Fake Profile Detection System

A Machine Learning based web application that detects fake Instagram profiles using profile metadata.

## Features

- Detects fake and genuine Instagram profiles
- Random Forest Machine Learning Model
- Flask-based Web Application
- Real-time Prediction System

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Flask
- HTML

## Dataset Features

- Profile Picture
- Username Statistics
- Full Name Statistics
- Description Length
- External URL
- Privacy Status
- Number of Posts
- Followers Count
- Following Count

## Model Performance

Accuracy: 98.9%

## How to Run

```bash
pip install pandas scikit-learn flask
python train.py
python app.py
```

Open:

http://127.0.0.1:5000
